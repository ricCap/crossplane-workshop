# Journeys — Overview

A journey is a cross-cutting suggestion thread for a real-world scenario: a coherent combination of 2XX, 3XX, and 4XX tasks you'd compose to build something end-to-end. Journeys aren't tasks themselves — they frame a scenario and point at the tasks that fit.
