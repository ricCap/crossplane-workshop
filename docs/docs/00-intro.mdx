# Welcome to the Crossplane Workshop

This workshop guides you through using [Crossplane](https://crossplane.io) to provision and manage
cloud infrastructure using Kubernetes-native APIs.

## Your environment

Each participant pair has an isolated [vCluster](https://www.vcluster.com) sandbox on the shared
management cluster. Your pair ID is embedded in this page's URL: `/p/<pair-id>/...`.

- **Your vcluster**: `<pair-id>` in namespace `participant-<pair-id>` on the management cluster
- **What you own**: everything inside that vcluster
- **What you share**: nothing — other pairs cannot see or affect your resources

## Workshop format

Each module ends with an interactive check. When a check turns green, the tutorial knows your
vcluster is in the expected state and you can move on.

Ready? Start with module 1.
