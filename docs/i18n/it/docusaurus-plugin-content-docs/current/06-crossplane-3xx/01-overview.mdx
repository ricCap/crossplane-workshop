# Crossplane 3XX — Panoramica

Task pointer-driven di difficoltà media per coppie a loro agio con provider e Composition. Ogni modulo inquadra un problema, elenca le pagine upstream che vale la pena leggere, e lascia il passo-passo a te.

## Moduli disponibili

| Modulo | ⏱️ | Argomento |
|---|---|---|
| [Pacchettizza come Configuration](./configurations) | 15m | Impacchetta gli XRD e le Composition che hai scritto in 101 in un artifact OCI versionato (un package `Configuration`), pinna le sue dipendenze provider/function, pubblicalo su un registry, installalo su un cluster fresco con un solo `kubectl apply`. |
| [provider-github contro un'org reale](./provider-github) | 20m | Installa `provider-github`, collegalo a una GitHub App pre-staged (o al tuo PAT fine-grained in modalità solo) e riconcilia un `Repository` reale nella sandbox condivisa `riccap-demo-org`. |
| [provider-aws contro il tuo account AWS](./provider-aws) | 35m | Iscriviti al free tier di AWS, crea un IAM user con accesso S3, installa `provider-aws-s3`, collega un `ClusterProviderConfig` e crea una sola MR `Bucket` S3. |
| [provider-gcp contro il tuo progetto Google Cloud](./provider-gcp) | 35m | Iscriviti a Google Cloud, crea un progetto, crea un service account con `roles/storage.admin`, installa `provider-gcp-storage` e crea una sola MR `Bucket` GCS. |
| [provider-azure contro la tua subscription Azure](./provider-azure) | 35m | Iscriviti ad Azure, crea un resource group, crea un service principal limitato ad esso, installa `provider-azure-storage` e crea una sola MR Storage `Account`. |

Altri task atterrano qui man mano che vengono scritti. Roadmap della sezione (per la [style guide](../)): lo stack dei secret, le status function, le strategie di drift remediation e altri argomenti "hai le basi — ecco com'è la produzione".

Il catalogo completo dei provider (con versioni correnti e schema browser) vive sul **[Crossplane Marketplace](https://marketplace.upbound.io/providers)**. I tre moduli cloud-provider qui sopra pinnano versioni specifiche per mantenere i comandi riproducibili; controlla il marketplace quando vuoi una release più nuova o un servizio diverso della stessa famiglia.
