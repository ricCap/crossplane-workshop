# Crossplane 4XX — Panoramica

Filoni avanzati per coppie pronte a uscire dai binari. Argomenti come generare provider da Terraform con Upjet o aggiornare i provider da v1 a v2. Per lo più pointer alla documentazione upstream — tracci tu il tuo percorso.

## Moduli disponibili

| Modulo | ⏱️ | Argomento |
|---|---|---|
| [Scrivi un provider con Upjet](./author-upjet-provider) | 30m | Crea uno scaffold di un provider Crossplane partendo da `upjet-provider-template`, puntalo a un qualsiasi provider Terraform del registry, scrivi le configurazioni di external-name e di gruppo per una o due risorse, lancia `make generate`, e riconcilia una vera Managed Resource sul tuo cluster del workshop. |

Altri task atterreranno qui man mano che vengono scritti — migrazione provider v1→v2, Composition Functions personalizzate in Go, e altri argomenti "ormai i binari ti stanno stretti".
